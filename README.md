# usb-reset

Linux system utility for usb port reset.

## Dependencies

- [libusb](https://libusb.info/)

## Install

After clone the project:

```sh
cd usb-reset
sudo make install
```

## Usage

In this version enter the following command after installation

```sh
sudo usb-reset
```
